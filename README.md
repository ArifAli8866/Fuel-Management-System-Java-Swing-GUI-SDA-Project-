# ⛽ Fuel Management System (Java Swing GUI)

## 📌 Project Overview
This is a Java-based Fuel Management System developed as part of the Software Design & Analysis (SDA) course.  
The project demonstrates implementation of UML design into a working GUI application using Java Swing.

---

## 🎯 Features

### 👤 Authentication System
- Login for Admin and User (Driver)

### 🚗 Fuel Request Module
- Users can submit fuel requests
- Enter vehicle ID and fuel quantity

### 🛠 Admin Panel
- View all fuel requests
- Approve pending requests
